# Velkyn Dink configuration

## Installation: 

- Install Dink plugin for RuneLite
- Open Dink settings and open the Advanced tab on the bottom
- For "Dynamic Config URL", fill in: `https://raw.githubusercontent.com/VelkynOSRS/DinkConfig/refs/heads/main/dink.json`
- For "Import Policy", CTRL+click both "Override Webhooks" and "Override Item Lists"
- Restart the RuneLite client

## Update notes

When making an update, make sure to set the correct version in the `embedFooterText` property of the configuration.
